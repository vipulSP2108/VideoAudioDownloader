# Streamlit YouTube Processor

A prototype web application built with Streamlit to process YouTube URLs, demonstrating UI design, user workflows, queue handling, and media processing concepts.

## Setup

1. Ensure you have Python 3.8+ installed.
2. Clone this repository or download the files.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application locally:
   ```bash
   streamlit run app.py
   ```

## Disclaimer
This project is created for educational purposes. It demonstrates Python application development, Streamlit interfaces, queue management, and media-processing workflows. Users should only process content they have permission to access. The project is intended as a learning prototype and not as a service for unauthorized content distribution.
